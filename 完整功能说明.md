## 📊 数据流向

```
【周一早上】你的操作
├─ 从飞书导出Excel
├─ 发给我：Excel + 范画图片文件夹路径

├─ 5维度评分：切题、示范、艺术、完整、创意

【你上传到Dashboard】
上传增强版Excel → 系统自动：
├─ 总览看板：显示合格率、不合格率
├─ 学区统计：计算各学区的合格率、优秀率
├─ 课程详情&范画：按评分排序显示所有范画
└─ 范画总体点评：自动筛选≥4.5分优秀范画
```

## 🎯 各标签页功能说明

### **📊 总览看板**
```
功能：整体数据概览
显示：
├─ 6个统计卡片（原4个 + 新增2个）
├─ 学区分组统计表（原9列 + 新增3列AI指标）
└─ 图表对比（学区提交率、课程提交情况）

新增AI指标：
├─ 范画合格率：AI评分≥3.0的比例
├─ 范画不合格率：AI评分<3.0或=0的比例
└─ 各学区的合格率、不合格率、优秀率
```

### **📚 课程详情&范画**
```
功能：查看每门课程的所有范画
操作：点击课程名 → 显示该课程的所有范画
排序：自动按AI评分从高到低
显示：所有范画（不筛选），包括优秀、合格、问题作品

示例：
点击"系统课L5窗边风景"后：

[西柠 ⭐4.8]  [PuDing ⭐4.3]  [童言 ⭐4.1]
优秀          合格            合格

[知夏 ⭐3.8]  [小圆 ⭐3.7]  [辣辣子 ⭐3.4]
合格          合格          合格

[遥愈 ⚠️0]   [粟米 ⚠️0]    [南茜 ⚠️0]
抄袭          抄袭          技法不符
```

### **📝 范画总体点评**
```
功能：展示优秀范画集锦
默认：只显示≥4.5分的优秀范画
筛选：
├─ 课程筛选：选择特定课程
├─ 学区筛选：选择特定学区
├─ 等级筛选：优秀（≥4.5）/合格（≥3.0）/问题（=0）
└─ 排序方式：评分高低/按课程分组/按姓名

特色功能：
├─ 统计卡片：实时显示筛选后的数据
├─ 评比画廊：卡片展示，点击查看详情
├─ 抄袭检测：自动列出所有问题作品
└─ 按课程分组：方便对比不同课程的优秀作品
```

## 💡 使用场景

### **场景1：查看整体情况**
```
目的：了解本周备课整体质量
操作：
1. 打开dashboard → 总览看板
2. 查看：范画合格率、不合格率
3. 对比：各学区的优秀率排名
```

### **场景2：查看某门课程的所有范画**
```
目的：查看"系统课L5窗边风景"的所有老师提交情况
操作：
1. 打开dashboard → 课程详情&范画
2. 点击"系统课L5窗边风景"
3. 自动按评分排序显示所有范画（优秀+合格+问题）
```

### **场景3：选出优秀范画展示**
```
目的：选出本周最优秀的范画用于表彰或展示
操作：
1. 打开dashboard → 范画总体点评
2. 默认显示所有≥4.5分的优秀范画
3. 可选：筛选特定课程或学区
4. 可选：按课程分组查看
```

### **场景4：识别问题作品**
```
目的：查看抄袭、技法不符的问题作品
操作：
1. 打开dashboard → 范画总体点评
2. 等级筛选：选择"问题作品"
3. 查看：抄袭检测列出所有0分作品
4. 联系相关老师处理
```

## 🎯 评分标准

| 等级 | 分数范围 | 说明 |
|------|----------|------|
| **优秀** ⭐ | ≥4.5分 | 在"范画总体点评"默认展示 |
| **合格** ✓ | 3.0-4.4分 | 符合教学要求 |
| **待改进** ⚠️ | 0.1-2.9分 | 需要提升 |
| **问题作品** ❌ | 0分 | 抄袭/技法不符/重复提交 |
