# 📷 范画图片展示功能说明

## 🎯 图片自动匹配规则

Dashboard会自动尝试加载图片，按以下优先级：

### **规则1：自动拼接路径（推荐）**

```
images/{教师艺名}_{课程简称}.jpg
```

**课程简称规则：**
- 自动去除"系统课"、"体验课"、"L数字"前缀
- 示例：
  - `系统课L5窗边风景` → `窗边风景`
  - `系统课L2许愿清单` → `许愿清单`
  - `《热观世界》` → `热观世界`

**命名示例：**
```
爱神_热观世界.jpg          ← 教师：爱神，课程：《热观世界》
西柠_窗边风景.jpg          ← 教师：西柠，课程：系统课L5窗边风景
童言_窗边风景.jpg
夏天老师_许愿清单.jpg
```

**支持的格式：** `.jpg`, `.jpeg`, `.png`

### **规则2：简化命名（备选）**

如果一个老师只有一张范画：
```
images/{教师艺名}.jpg
```

示例：`爱神.jpg`

## 📂 使用步骤

### **步骤1：准备图片**

1. 把所有范画图片重命名为规则格式
2. 确保文件名不含特殊字符（只用中文、英文、数字、下划线）

### **步骤2：上传到GitHub**

#### **方法A：通过Git命令（推荐）**

```powershell
# 1. 把图片复制到 images/ 目录
Copy-Item "你的图片文件夹\*.jpg" "C:\Users\xulaixiang\WorkBuddy\2026-06-15-16-40-32\deploy\images\"

# 2. 进入项目目录
cd C:\Users\xulaixiang\WorkBuddy\2026-06-15-16-40-32\deploy

# 3. 提交并推送
git add images/
git commit -m "上传本周范画图片"
git push origin main

# 等待1-2分钟，GitHub Pages自动部署
```

#### **方法B：通过GitHub网页**

1. 打开：https://github.com/laixiang-xu/beike-dashboard/tree/main
2. 点击"Add file" → "Upload files"
3. 选择或拖拽图片文件
4. 写提交信息："上传本周范画图片"
5. 点击"Commit changes"

### **步骤3：访问Dashboard验证**

1. 访问：https://laixiang-xu.github.io/beike-dashboard/
2. 上传包含AI评分的Excel
3. 点击课程 → 看到范画图片自动显示

## 🆘 常见问题

**Q1: 图片显示"图片未上传"？**
- 检查文件名是否完全匹配（区分大小写）
- 检查文件是否已push到GitHub
- 等待1-2分钟GitHub Pages部署

**Q2: 课程名太长怎么办？**
- 手动简化命名
- 例如：`系统课L5窗边的美好时光` → 用 `西柠_窗边.jpg`

**Q3: 一个老师有多门课的范画？**
- 每门课一个文件：
  ```
  西柠_窗边风景.jpg
  西柠_许愿清单.jpg
  ```

**Q4: 图片文件太大？**
- 建议压缩到500KB以内（在线工具：tinypng.com）
- 尺寸建议：宽度800-1200px

**Q5: 能用中文文件名吗？**
- 可以！完全支持中文

**Q6: 本地测试看不到图片？**
- 本地文件路径和GitHub路径不同
- 需要推送到GitHub后通过网页访问
